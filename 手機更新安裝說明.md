# 沖繩行程｜手機新增 v23

這版使用全新的 Workflow 檔名，避開舊 `mobile-add.yml` 的「手動停用」狀態。

## 安裝

1. 將本資料夾內所有檔案與資料夾上傳至 `chenjiunli/okinawa-trip` 的 `main` 分支，保留 `.github`、`data`、`scripts` 結構並取代 `index.html`。
2. 舊的 `.github/workflows/mobile-add.yml` 可以留著，不影響新版；之後有空再刪除即可。
3. 到 `Settings → Actions → General`：
   - `Actions permissions` 選 `Allow all actions and reusable workflows`。
   - `Workflow permissions` 選 `Read and write permissions`。
   - 按 `Save`。

## 手機新增

1. 開啟行程網頁，按「＋ 新增景點／餐廳」。
2. 新頁面標題應為「手機新增景點餐廳 v23」。
3. 按 `Run workflow`、填寫資料，再按綠色 `Run workflow`。
4. 約數分鐘後重新整理行程網頁。

如果仍顯示停用，代表整個儲存庫的 Actions 尚未開放，而不是 Workflow 檔案錯誤。
