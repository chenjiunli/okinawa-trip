# 沖繩行程｜手機新增測試版

## 第一次安裝

1. 將本資料夾內的所有檔案與資料夾上傳到 `chenjiunli/okinawa-trip` 的 `main` 分支，保留 `.github`、`data`、`scripts` 資料夾結構。
2. 到 GitHub 儲存庫：`Settings → Actions → General`。
3. 在 `Workflow permissions` 選擇 `Read and write permissions`，按 `Save`。
4. 等 GitHub Pages 重新發布後，開啟原本的行程網頁。

## 手機新增

1. 在行程網頁按「＋ 新增景點／餐廳」。
2. 登入 GitHub，按 `Run workflow`。
3. 填寫日期、類別、名稱、介紹、Google Maps 分享連結；照片與 IG 可留空。
4. 再按一次綠色 `Run workflow`。
5. 約數分鐘後重新整理行程網頁，新項目會出現在指定日期的建議景點區。

## 注意

- 目前版本先支援「新增」；同一天輸入相同名稱會更新原項目，避免重複。
- 只有儲存庫擁有者可以執行新增流程。
- Google Maps 短連結若能解析座標，新增項目也會自動出現在全程地圖。
- 不要把 GitHub 密碼或 Personal Access Token 填進表單。
